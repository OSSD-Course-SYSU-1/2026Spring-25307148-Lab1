# 新增功能：冒泡排序分步可视化

## 一、功能简介
在原有算法展示项目的基础上，新增了冒泡排序的**分步可视化**演示模块。该功能将冒泡排序的每次比较和交换过程，通过彩色柱状图和动态文字实时呈现，极大提升了算法的直观性和教学效果。

## 二、新增文件
| 文件路径 | 作用 |
|----------|------|
| entry/src/main/ets/pages/BubbleSortVisual.ets | 可视化排序页面，包含动画逻辑 |

## 三、修改文件
| 文件路径 | 修改内容 |
|----------|----------|
| entry/src/main/resources/base/profile/main_pages.json | 注册新页面路由 `pages/BubbleSortVisual` |
| entry/src/main/ets/pages/Index.ets | 在算法列表中添加“冒泡排序可视化”入口 |

## 四、核心技术点
1. **@State 状态管理**：通过 `@State` 修饰 `array`、`compareIndex`、`sortedBoundary`，实现数据与 UI 的自动同步。
2. **异步动画控制**：利用 `async/await` 和 `setTimeout` 封装 `delay` 函数，在排序循环中暂停执行，制造逐步动画效果。
3. **动态视觉反馈**：
    - 正在比较的两个元素用**红色**高亮。
    - 已完成排序的区域用**绿色**标识。
    - 未排序区域为**蓝色**。
4. **健壮性处理**：`isRunning` 锁防止重复点击；`clearTimeout` 避免内存泄漏。

## 五、运行效果说明
- 点击“生成数组”生成随机高度的柱状图（8 个元素）。
- 点击“开始排序”，柱子逐步比较、交换，颜色实时变化。
- 底部文字提示当前操作（比较/交换）。
- 排序完成后所有柱子变为绿色，提示“排序完成！”。
